# thirdweb NFTDrop nextjs templates

This templates demonstrate the frontend connecting NFTDrop thirdweb's pre-build contract.

### Install

```
yarn install
```

### Run

```
yarn dev
```

Open http://localhost:3000 with your browser to see the result.
